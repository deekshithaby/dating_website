💘 ShowUp — Dating App: Project Brief for Claude Code
🧠 One-Line Concept

"No swiping. Real matches. Show up or miss out."

A dating app where compatibility is algorithmic, profiles are hidden until both people physically show up to a real-world meeting, eliminating ghosting and fake profiles entirely.

🎯 Product Vision
ShowUp is a hybrid of a dating app + event booking + trust verification system. It removes the infinite-scroll dopamine loop of modern dating apps and replaces it with intentionality, accountability, and real-world outcomes.
The core mechanic: you only see each other's photo after both of you arrive at the same location at the same time.

🧩 Full App Flow
1. Landing Page

Bold, minimal hero with the tagline
Single CTA: "Get Started"
No distractions, no feature list — pure hook

2. Authentication

Phone number input
OTP verification (SMS)
Creates user account on success
Output: User is authenticated and inside the app

3. Onboarding — Phase 1
A. Personality & Compatibility Quiz
Fields to capture:

Personality type (introvert/extrovert, values, communication style)
Lifestyle (activity level, social habits, daily rhythm)
Dating intent (casual, serious, open to see)
Boundaries (dealbreakers, preferences)


This data feeds the matching algorithm.

B. Photo Upload

Clear face photo required (no sunglasses, masks)
Photo is hidden from matches until GPS verification at meetup
Used for trust/anti-fake-profile layer
Optional: ID or selfie verification (Phase 2)

C. Completion Screen

Animated loader: "Finding your matches…"
User is now in the matching pool


4. Matching System — Phase 2
Filtering logic (Phase 1 / MVP):

Age range
City / proximity
Dating intent alignment

Compatibility scoring (Phase 2 / AI layer):

Quiz response similarity
Behavioral signals

Unique Mechanic 💡

A girl's profile is sent to multiple compatible guys simultaneously
The first guy who pays gets the match locked in
Creates urgency, reduces passive ghosting behavior

Output: A confirmed match with date/time/location details

5. Match Reveal Flow — Phase 3
Once matched:

Limited chat unlocked (optional, time-gated before meetup)
Match screen shows:

Meeting location (pre-selected or user-suggested)
Date & time
Countdown timer


Reminder notifications sent: 24h, 1h, 15min before meetup
Instruction: "Keep GPS on when you arrive"


6. Real-World Verification — Phase 4 (Core USP 🚀)
At the time of the meeting:

App checks GPS of both users
Confirms both are at the designated location
✅ If both present → Photo reveal unlocks
❌ If one or both don't show → No reveal, match voided

If girl doesn't show:

Guy receives 1 free match credit as compensation

Why this works:

No catfishing (photo hidden until verification)
No ghosting (financial + social commitment required)
Real accountability baked into the product mechanic
